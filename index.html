プライバシーポリシー

ゴールノート

1. はじめに

ゴールノート（以下「本アプリ」）は、お子様のサッカー活動（試合・練習・イベント）を記録・管理するためのiOSアプリです。 本プライバシーポリシーは、本アプリがどのような情報を収集・利用するかを説明します。

本アプリをご利用いただくことで、本ポリシーの内容に同意いただいたものとみなします。

2. 収集する情報

端末内のみに保存する情報（外部サーバーへは送信されません）

  • 選手名・チーム名・背番号・チームアイコン画像
  • イベント情報（日時・種別・対戦相手・メモ・会場・スコア）
  • タスク・準備リスト・試合スタッツ
  • 添付ファイル（画像・PDFなど）
  • 天気取得のデフォルト地点（地名・緯度・経度）

天気機能利用時に外部サービスへ送信する情報

  • 地点検索時の検索文字列（地名）― Open-Meteo Geocoding API へ送信
  • 天気取得時の緯度・経度 ― Open-Meteo Weather API へ送信

Open-Meteo（open-meteo.com）はオープンソースの気象データサービスです。送信した位置情報は天気データの取得のみに使用され、当方のサーバーには保存されません。

グループ共有機能利用時に当方サーバーへ送信する情報

  • 匿名ユーザーID（アプリが自動生成するランダムな識別子）
  • 表示名（ユーザーが任意で設定した名前）
  • 共有したイベント情報（タイトル・日時・開始時刻・種別・対戦相手・会場・メモ・スコア・天気地点名・緯度・経度など）

アプリ改善のために自動収集する情報（Firebase）

  • Firebase Analytics：画面表示・機能の利用状況、アプリインスタンスID、端末情報（OS・モデル）
  • Firebase Crashlytics：クラッシュ発生時のスタックトレース・端末情報・アプリ状態、Crashlytics インストールUUID

これらのデータはGoogle LLC（Firebase）のサーバーで処理されます。詳細はGoogleのプライバシーポリシーをご参照ください。

3. 位置情報の利用について

本アプリは天気情報の取得を目的として位置情報を使用します。 取得した位置情報（緯度・経度）は天気データ取得のためにOpen-Meteo APIへ送信されます。 当方のサーバーには保存されませんが、グループ共有機能をご利用の場合は、設定した天気地点情報（地名・緯度・経度）が共有イベントの一部として当方サーバーに送信・保存されます。

位置情報の利用にはiOSの許可が必要です。許可しない場合でも、本アプリの基本機能はご利用いただけます。

4. カメラ・写真・ファイルへのアクセスについて

以下の目的でデバイスへのアクセス許可をお願いする場合があります。

  • カメラ：イベントへの写真添付のために使用します
  • フォトライブラリ：チームアイコン設定やイベントへの写真添付のために使用します
  • ファイル（Files App）：イベントへのPDFなどのファイル添付のために使用します

取得した画像・ファイルはすべて端末内にのみ保存されます。

5. 情報の第三者提供

以下の場合を除き、収集した情報を第三者に販売・提供・開示することはありません。

  • 法令に基づき開示が必要な場合
  • Open-Meteo（天気・地点検索のためのAPI呼び出し）
  • Firebase／Google LLC（アプリ分析・クラッシュレポートのためのデータ処理）

6. データの保存・削除・安全管理

端末内のデータ：アプリをアンインストールすることで削除されます。ただし、iOS Keychainに保存されたユーザーIDはアンインストール後も端末に残る場合があります（「設定」→「一般」→「転送またはiPhoneをリセット」→「すべてのコンテンツと設定を消去」で削除されます）。

サーバー上のデータ：グループ共有機能を通じてサーバーに送信したデータの削除をご希望の場合は、下記お問い合わせ先にご連絡ください。速やかに対応します。

Firebaseのデータ：Firebase AnalyticsのデータはGoogleのAnalytics設定に従い保持・管理されます。Crashlyticsのクラッシュレポートは90日間保持されます。詳細はGoogleのデータ保持ポリシーをご参照ください。

通信の安全管理：本アプリと外部サービス間の通信はすべてHTTPS（TLS）で暗号化されています。

7. お子様のプライバシーについて

本アプリは保護者の方がお子様のサッカー活動を記録することを想定しています。 13歳未満のお子様本人がアプリを使用する場合は、保護者の監督のもとでご利用ください。 意図せずお子様の個人情報を収集していることが判明した場合は、速やかに削除します。

8. ポリシーの変更

本プライバシーポリシーは必要に応じて更新する場合があります。 重要な変更がある場合はアプリ内でお知らせします。 最新版は常にこのページでご確認いただけます。
